The CPU - Computer's Brain

The CPU is like the brain of the computer. It does all the thinking and solving problems. It follows instructions very fast and helps run all your favorite games and apps!

Memory (RAM)

Memory is where the computer keeps things it's working on right now. It's like your desk where you put things you're using. When you turn off the computer, everything in memory goes away!

Storage

Storage is where all your files, games, and pictures are kept safe. It's like a big bookshelf where you can store things for a long time, even when the computer is turned off!

Input & Output Devices

Computers need special devices to talk to us and for us to talk to them! Input devices help us send information TO the computer. Output devices help the computer send information TO us!
